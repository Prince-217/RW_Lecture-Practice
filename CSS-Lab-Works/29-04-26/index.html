Flex - Box Properties

1. flex-grow : (amount);

Theory

Definition and Usage

The CSS flex-grow property defines how a flex item expands to fill available space within a container relative to its siblings. It accepts a unitless number (default is 0) that acts as a proportion, where a higher number allows the item to grow more. It is essential for creating responsive layouts that fill extra screen space.

Ex: .div-1{ flex-grow: 4; }
